pug Tips

タグとテキスト

p テキスト
ul: li リスト
p あい#[strong う]えお
<p>テキスト</p>
<ul><li>リスト</li></ul>
<p>あい<b>う</b>えお</p>

複数行テキスト

p
	| 複数行のテキスト
	br
	| インデントは自由
<p>
	複数行のテキスト<br>
	インデントは自由
</p>

クラスとID

p.test.red
h1#sitename
<P class="test red"></p>
<h1 id="sitename"></h1>

divの省略

#app
.test
<div id="app">
<div class="test">

エスケープ

code= '<p>あ</p>\n<i>い</i>'
<code><p>あ</p>
<i>い</i></code>

属性(プロパティ)

input(type="radio"
	name="test"
	checked)
<input type="radio" name="test" checked>

scriptやstyle

style.
	a {color: red}
script.
	let pug = "Hello, pug!"
<style>
	a {color: red}
</style>
<script>
	let pug ="Hello, pug!"
</script>

変数

- var pagename="ページ"

title #{pagename}
<title>ページ</title>

変数:プロパティ

- var clsname="aaa"

p(class=clsname)
p(class=`&{clsname}`)
<p class="aaa"></p>
<p class="aaa"></p>

for

- for (var x=0; x<3; x++)

li ナンバー#{x}
<li>ナンバー0</li>
<li>ナンバー1</li>
<li>ナンバー2</li>

each

each value in
 ["HTML","CSS","JS"]

li #{value}
<li>HTML</li>
<li>CSS</li>
<li>JS</li>

if

- var name = "うさ"
if name=="やぎ"
	p わたしはやぎです
else if name=="ヒツジ"
	p わたしはヒツジです
else
	p やぎでもヒツジでもない？
<p>やぎでもヒツジでもない？</p>

switch

- var name = "うさ"
case name
	when "やぎ"
		p わたしはやぎです
		break
	when "ヒツジ"
		p わたしはヒツジです
		break
	default
		p やぎでもヒツジでもない？
<p>やぎでもヒツジでもない？</p>
