式と演算子

等価演算子 == or !=

1 == "1" // true
0 == false // true
"a" != "a" // false
1 != "1" // false

OR(||) - 片方がtrueならばtrue

10 < 20 || "Hello" == "こんにちは"
// true

AND(&&) - 両方がtrueならばtrue

10 < 20 && "Hello" == "Hello"
// true
