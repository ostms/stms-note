エスケープ

<code><p>あ</p>
<i>い</i></code>
  • タグ名の右横に=をつけて、''または""で内容を囲む

テンプレート

extends 🔗

_layout.pug

body
	block content
		h1 デフォルトタイトル

index.pug

extends _layout.pug
block content
	h1 サイト名
<body>
	<h1>サイト名</h1>
</body>
  • extends は 一番上に記述しないとエラーが出る。ページ構造全体を読み込むものだからかな。
テンプレート

block option

_layout.pug

block section
	h2 セクションタイトル
	p テキスト１テキスト１

page.pug

block append section
	p 追加の内容
<h2>セクションタイトル</h2>
<p>テキスト１テキスト１</p>
<p>追加の内容</p>
  • blockは呼び出されると中身が全て上書きされる（消える）仕様だが、block append **と書くことで、blockの中身を継承しながら呼び出し先で追記することができる。
append
blockの最後尾に記述した内容を追加する
prepend
blockの先頭に記述した内容を追加する

mixin list(url,title)

		li: a(href=`${url}`) #{title}

+list("./",HOME)

	<li><a href="./">HOME</a></li>
</ul>
